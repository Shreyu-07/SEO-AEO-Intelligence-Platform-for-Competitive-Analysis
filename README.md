## 🛰️ Overview

The **SEO & AEO Intelligence Platform** is a high-performance auditing engine designed for the modern semantic web. While traditional tools focus solely on search engine rankings, this platform prioritizes **Answer Engine Optimization (AEO)**—ensuring your assets are prioritized by LLMs and Neural Search Engines (like Perplexity, ChatGPT, and Gemini).

## 💎 Key Capabilities

- **Neural Scraping**: Proprietary DOM traversal optimized for JS-heavy, reactive environments.
- **AEO Answerability Index**: Rigorous evaluation of content clarity for AI model consumption.
- **Technical SEO Audit**: Deep-scan of metadata, heading architecture, canonical health, and accessibility signals.
- **Strategic SWOT Mapping**: AI-driven opportunity identification against market competitors.
- **Secure Protocol**: Stateless audit architecture ensuring zero data leakage of proprietary URLs.

## 🛠️ Tech Stack

- **Frontend**: React 19 + Vite (HMR-enabled)
- **Styling**: Tailwind CSS (Monochrome High-Contrast UI)
- **Intelligence**: Google Gemini 2.5 Flash API
- **Scraping**: Puppeteer-core + @sparticuz/chromium-min
- **Runtime**: Vercel Serverless (Node.js)

## 🚀 Rapid Deployment

This project is optimized for a **single-target deployment on Vercel**.

1. **Fork/Clone** the repository.
2. **Import to Vercel**.
3. **Set Environment Variable**:
   - `GEMINI_API_KEY`: Your Google Gemini API Key.
4. **Deploy**.

For detailed deployment instructions, see [DEPLOYMENT.md](./DEPLOYMENT.md).

## 💻 Local Setup

```bash
# Install dependencies
npm install

# Set up local environment
echo "GEMINI_API_KEY=your_key_here" > .env

# Run development server
npm run dev
```
